ForCasT

The CasComputer-AidedDesign, or ForCasT, tool provides functionality to design, view, and evaluate guides and primers for CRISPR mutagenesis experiments in model organisms. It integrates existing tools including JBrowse, BWA, bedtools, SAMtools, Primer3, BLAST, and Silica to create a complete pipeline. Select a genome from the list below and click 'Go' to get started.
